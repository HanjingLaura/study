# Flutter
## Flutter项目
- Flutter项目
    - 创建Flutter项目
        - flutter create --platforms web <项目名称>
    - 运行Flutter项目
        - lib/main.dart
        - 点击Run
- Flutter项目目录
    - 解析工程目录结构
        - .开头的文件：多为编译器/工具自动生成
        - lib：项目的主要源代码目录
            - main.dart：应用程序入口点
        - test：测试文件目录
        - web：Web平台特定的配置和资源文件
        - pubspec.yaml：项目依赖和配置文件
