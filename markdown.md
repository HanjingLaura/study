# Markdown 语法全总结 (Cheat Sheet)

---

## 1. 标题 (Headers)
使用 `#` 号标记，一级到六级标题对应 `#` 的数量。
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

---

## 2. 文本样式 (Emphasis)
* **加粗**: `**加粗文本**` 或 `__加粗文本__` -> **加粗文本**
* **斜体**: `*斜体文本*` 或 `_斜体文本_` -> *斜体文本*
* **加粗并斜体**: `***加粗斜体***` -> ***加粗斜体***
* **删除线**: `~~删除线文本~~` -> ~~删除线文本~~

---

## 3. 列表 (Lists)
### 3.1 无序列表
使用 `-`、`+` 或 `*` 加上空格。
- 项目 1
- 项目 2
  - 子项目 2.1 (前面加两个或四个空格)

### 3.2 有序列表
使用数字加上点和空格。
1. 第一步
2. 第二步

### 3.3 任务列表 (Checkboxes)
- [x] 已完成任务
- [ ] 未完成任务

---

## 4. 链接与图片 (Links & Images)
* **超链接**: `[链接文字](链接地址 "可选标题")`
  [百度](https://www.baidu.com)
* **引用链接**: `<https://www.google.com>`
* **插入图片**: `![图片描述](图片链接或路径)`
  ![示例图片](https://via.placeholder.com/150)

---

## 5. 代码 (Code)
### 5.1 行内代码
使用单个反引号包裹。
使用 `library(shiny)` 加载包。

### 5.2 代码块
使用三个反引号包裹，并可以指定语言进行高亮。
```R
# 这是一个 R 代码块
server <- function(input, output) {
  print("Hello Shiny")
}
```

---

## 6. 引用 (Blockquotes)
使用 `>` 符号。
> 这是一个引用段落。
>> 这是嵌套引用。

---

## 7. 表格 (Tables)
使用 `|` 分隔列，使用 `-` 分隔表头。
| 字母 | 数字 | 字符串 |
| :--- | :--: | ---: |
| a | 1 | aaa111 |
| b | 2 | bbb222 |
| c | 3 | ccc333 |

---

## 8. 其他常用语法
* **分割线**: 使用三个或以上的 `---`、`***` 或 `___`。
* **换行**: 行末加两个空格并回车，或者直接空一行（推荐）。
* **转义**: 如果想显示特殊字符（如 `#`），在前面加反斜杠 `\`。
  \# 这不是标题，而是一个普通的井号。

---

## 9. 扩展语法 (部分编辑器支持)
* **数学公式**: 
  行内公式: $E = mc^2$
  块状公式:
  $$ \sum_{i=1}^{n} i = \frac{n(n+1)}{2} $$
* **脚注**: `这是一个脚注[^1]`
  [^1]: 这里是脚注的具体内容。
